                                                                        |
- - - - - - - - - - |Informacje o Wybrajen Stacji | - - - - - - - - - - |
2. WYBRANA STACJA                                                       |
-Wyświetlany jest pełen adres wybranej stacji.                          |
                                                                        |
3. WYBRANA STACJA WYKONUJE POMIARY                                      |
-Wyświetlane są rodzaje zanieczyszczenia w powietrzu                    |
mierzone przez wybraną stację.                                          |
                                                                        |
  - - - - - - - - - - |Wybór wyświetlanych danych|- - - - - - - - - - - |
                                                                        |
4. ZAKRES DANYCH                                                        |
-Zaznacz z jakiego okresu chcesz wyświetlić dane                        |
                                                                        |
5. DANE DLA POMIARU                                                     |
-Zaznacz jaki rodzaj zanieczyszczenia wyświetlić                        |
                                                                        |
 - - - - - - - - - - - -|Dane Pomiarów Powietrza| - - - - - - - - - - - |
                                                                        |
6. STATYSTYKI                                                           |
-Wartości minimalna i maksymalna z zaznaczonego okresu                  |
                                                                        |
7. WYKRES POMIARÓW						                                        	|
-Zestawienie dostępnych pomiarów w wykresie                             |
(jednostką zanieczyszczenia jest mikrogram/m^3)                         |
                                                                        |
 - - - - - - - - - - - - |Informacje Dodatkowe|- - - - - - - - - - - - -|
                                                                        |
Wybranie stacji pomiarowej, powoduje pobranie informacji o zbieranych   |
przez nią danych. (sensors.json)                                        |
                                                                        |
Wybranie dla danej stacji rodzaju mierzonych danych, powoduje pobranie  |
informacji o tym rodzaju zanieczyszczenia powietrza. Ilość pobranych    |
pomiarów zależy od okresu pomiaru samej stacji. Możliwe do pobrania     |
na raz są dostępne w najlepszym przypadku ostatnie 24h z okresem 1h     |
                                                                        |
Baza pobranych danych przechowywana jest w measurements_history.json    |
